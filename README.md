# SmartRetail: Predictive Analytics for Business Intelligence 📊

A comprehensive Data Science project developed to solve real-world retail challenges using synthetic datasets. This project focuses on predicting customer purchasing behavior and estimating monthly profit margins through machine learning.

##  Project Overview
The "SmartRetail" project follows a complete data analytics lifecycle:
* **Data Preprocessing:** Handling missing values and feature scaling.
* **Classification:** Predicting if a customer will make a purchase.
* **Regression:** Estimating the potential profit (Utility) for the upcoming month.

##  Key Technical Features
* **Automated Pipelines:** Implementation of Scikit-Learn `Pipeline` and `ColumnTransformer` to ensure clean, reproducible, and leak-free data processing.
* **Model Evaluation:** * **Classification:** Decision Tree Classifier with a deep analysis of the **Confusion Matrix** to identify class imbalance and improve business decision-making.
    * **Regression:** Comparative analysis between **Linear Regression** and **Decision Tree Regressor**, utilizing $R^2$ scores to determine the best fit for linear synthetic data.
* **Business Insights:** Strategic recommendations based on model results, focusing on customer retention and loyalty programs.

##  Technical Stack
* **Language:** Python 3
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn.
* **Environment:** Jupyter Notebook.

## 📁 Project Structure
* `Proyecto_SmartRetail_GonzalezHeredia.ipynb`: The main notebook containing the full analysis, visualizations, and model training.
